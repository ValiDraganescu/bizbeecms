# CMS v1.22.0

_2026-07-06 · changes since r-1.21.0_

## Features
- Components gallery: a redesigned components view. The old fixed starter kits (blog / docs / landing / portfolio / pricing) are gone — instead you export and import component kits as a single `.zip` (component definitions plus their asset bytes), so you can move a curated set between Sites.
- Google Fonts in theme settings: a fonts editor lets you pick Google fonts for the Site; they're wired into the rendered pages.
- Settings restructure: settings now has a proper landing page and a shared layout across the settings sections.

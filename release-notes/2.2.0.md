# CMS v2.2.0

_2026-07-23 · changes since r-2.1.1_

## Features
- **Assistant chat history.** The CMS AI assistant now saves your conversations. A new *Settings → Assistant chats* page shows daily usage and every stored transcript — with all messages and tool calls — ready to review, download, or delete. Your saved threads carry over between sessions.
- **Delete a Site.** You can now delete a Site from ProjectManager. Deletion tears down the Site's deployed Cloudflare resources — its Worker, database, media bucket, and any custom domains — derived from the actually-deployed name so a later slug rename can't leave resources behind.

# CMS v1.9.0

_2026-06-26 · changes since r-1.8.0_

## Features
- Chat attachments: attach images/files to a chat message via a picker or drag-and-drop, gated to models that support them; attachments are sent inline to the assistant.
- Component kits: preview a kit's contents before installing (uploaded kits included), see per-component install results, and add a name + description to a kit on export.
- Component tags: bulk-edit tags across components, and group the page-builder component rail by tag.
- Page builder: collapsible left rail and right inspector, plus a resizable inspector with three preset widths.
- MCP: the CMS advertises its MCP URL from the site's configured custom domain (via APP_ORIGIN) instead of the request host, fixing the URL on custom domains.

## Fixes
- MCP URL was derived from the incoming request host, which broke on custom domains — now uses the site's APP_ORIGIN.

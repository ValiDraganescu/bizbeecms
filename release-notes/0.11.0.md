# CMS v0.11.0

_2026-06-24 · changes since cms-v0.10.0_

## Features
- Collections: import and export items as CSV or JSON, from a new operator UI.
- Collections: a schema manager to add, drop, and rename fields, with atomic live-store rebuilds.
- Collections: AI tools to drop and rename collection fields (schema evolution).
- Collections: an operator raw-SELECT SQL console (fenced SELECT-only, not exposed to the AI).
- AI chat panel resizes via a top-left pointer-drag handle (replacing the native CSS resize).
- Chat: scroll-to-latest pill with follow-bottom-only transcript behavior.
- Chat: tool cards get a show-more toggle for truncated input/output blobs.
- Chat: inline active-override label near the input (PM-SSO).
- Accessibility: focus trap and initial focus in the open chat dialog, Esc minimizes the panel, and focus-visible rings on widget buttons.

## Fixes
- Collections: fix collection-create failing when the CREATE TABLE spanned multiple lines (use prepared statement instead of D1 exec).
- Collections: restore the missing Collections nav-item icon.
- Chat: fix the model-picker crash on stale-cache rows from /api/chat/models.
- Chat: fix the expand/shrink toggle getting stuck one-way (toggle on size, not preset).

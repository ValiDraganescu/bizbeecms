# CMS v2.3.1

_2026-07-24 · changes since r-2.3.0_

## Fixes
- **AI translation no longer hangs forever.** If the translation model opened its response but then went silent, the Page Builder field's translate spinner would spin indefinitely. It now times out with a clear, actionable message ("the model stopped responding — try again, or pick a different translation model in Settings → AI models") instead of hanging. Added on both the server (a 504 after the stream stalls) and the browser (a fetch timeout backstop), so the spinner always clears.

## Other
- Added structured, greppable logging to the translate endpoint (every step and failure mode) to make diagnosing model/streaming issues straightforward from Worker logs.

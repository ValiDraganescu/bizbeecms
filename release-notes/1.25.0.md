# CMS v1.25.0

_2026-07-07 · changes since r-1.24.0_

## Features
- Publish-together: when you publish a page that uses components with unpublished draft edits (for example an AI change that hasn't gone live), the CMS now prompts you to publish those draft components alongside the page — so the published page doesn't silently render stale component versions. Draft components are flagged with a badge in the Layers tree and inspector.

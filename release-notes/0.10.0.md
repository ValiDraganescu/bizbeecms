# CMS v0.10.0

_2026-06-24 · changes since cms-v0.9.2_

## Features
- AI chat assistant panel is now resizable, with default/half-screen preset toggles.
- Chat input is a resizable textarea with a switch for Enter behavior (send vs. newline).
- The model picker shows per-model prices (input/output USD per 1M tokens) and input-modality icons, and is filtered to tool-call-capable models only.
- Your selected model now persists across reloads (validated against the live catalog).
- The chat widget shows the remaining credit on the in-use OpenRouter key.
- Tool calls are persisted in chat history and rendered as cards (name shown once, with an input/output accordion).
- The launcher shows an unread badge when minimized and a new reply arrives.
- PM-SSO operators can export a chat as the full model-payload JSON.
- PM-SSO operators can edit system-prompt version overrides from the debug panel, backed by a new prompt-version store and gated prompts API.

## Fixes
- Accept 48-byte `CMS_AUTH_SECRET` values: the AES key is now SHA-256-derived instead of being rejected by the secret-box KEK.

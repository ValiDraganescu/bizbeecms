# CMS v1.1.0

_2026-06-25 · changes since r-1.0.1_

## Features
- **Page-aware AI assistant in the Page Builder.** When you have a page selected, your next message to the AI assistant automatically tells it which page you're editing (path, slug, draft/published), so page requests apply to the right page. It updates as you switch pages and clears when you leave the Page Builder.
- **"Page context attached" chip.** A small chip above the chat input shows when page context will be sent; click it to see the exact text being attached to your message.

## Other
- The AI assistant's debug (context & tools) button now uses a proper cog/settings icon instead of the previous sun-like one.

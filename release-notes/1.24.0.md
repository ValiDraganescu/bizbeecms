# CMS v1.24.0

_2026-07-07 · changes since r-1.23.0_

## Features
- Per-Site `robots.txt`: a settings editor with a safe default (crawlers allowed, `/admin` `/api` `/preview` disallowed) plus a free-text override and an auto `Sitemap:` pointer.
- IndexNow: publishing, unpublishing, deleting or renaming a page automatically notifies search engines of the changed URLs (per-Site key served at a fixed path).
- 301 redirects: auto-captured when a page is renamed, plus a manual redirects admin UI.
- Per-page SEO controls: a `noindex` toggle (also excludes the page from the sitemap and IndexNow), full Open Graph / Twitter cards, per-Site search-engine verification tokens (Google / Bing / Yandex), and automatic BreadcrumbList JSON-LD for nested pages.
- JSON-LD authoring: a JSON-LD component kind with a Develop editor (kind toggle, template editor, live JSON preview), List → schema.org `ItemList` aggregation, and an on-demand JSON-LD authoring guide for the AI.
- `/llms.txt` AI-crawler index and per-page `.md` variants for AI crawlers, with an editable `llms.txt` template (variable slots) and settings editor.
- SEO audit report: an admin page flagging orphan pages, broken internal links, missing meta, and missing image alt text — now scanning inside referenced component trees too. Plus AI tools to audit and bulk-fix missing meta and missing alt text.
- Responsive images: width variants served from `/media?w=`, `srcset`/`sizes` on rendered images, image dimensions captured at upload and stamped through for CLS stability (including AI-generated images).
- Open Graph images: auto-generated OG images with fallback serving and a regenerate button.
- A designated branded 404 page that renders in the visitor's URL locale, and a per-Site rate-limit threshold (Off / Normal / Strict) enforced at the edge on public page paths.

## Fixes
- Content-locales save now purges the sitemap and `/llms.txt` caches.

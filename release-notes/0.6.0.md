# CMS v0.6.0

_2026-06-22 · first tagged CMS release (baseline)_

This is the first `cms-v*` tag — the baseline release that PM can list and deploy.
It captures the AI-assistant CMS as it stands today.

## Features
- **AI assistant** — page-aware chat widget that creates content, authors custom UI
  components, composes pages, and translates, with a searchable model picker over the
  full Workers-AI catalog and per-Site history.
- **Page builder** — visual builder (layers / preview / block / page / SEO tabs),
  responsive columns with per-viewport visibility, per-locale SEO + OG image, and
  full versioning (draft / publish / history / restore) with AI-translate.
- **Custom components** — the AI emits a `{ tree, script, css }` artifact rendered
  server-side (data walk, no eval) with client scripts shipped to the browser, so
  custom interactive components run on Cloudflare Workers.
- **Content locales** — per-Site, data-driven user-facing content languages, distinct
  from the fixed EN/FI/ET admin UI.
- **Schema field types** — date/time props render as native inputs in component
  settings.

## Other
- Cloudflare-native throughout (Workers, D1, R2); no non-Cloudflare infra.
- Precompiled Tailwind utility sheet so runtime-artifact classes resolve.

# CMS v1.20.0

_2026-07-03 · changes since r-1.19.0_

## Features
- Site export / import: from Settings → Export / Import, download a whole Site as a single `.zip` (pages, components, collections and their items, settings, and asset bytes) and import it into another CMS instance. Import runs a validate + dry-run first, then a destructive replace, uploading the bundled asset bytes automatically.

## Fixes
- Export no longer silently truncates at 1000 rows — large collections export in full.

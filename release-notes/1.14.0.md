# CMS v1.14.0

_2026-06-29 · changes since r-1.13.1_

## Features
- Page builder: sections now have editable, inline-renamable names (auto-defaulting to "Section N").
- Chat: type `@` in the composer to autocomplete the current page's sections; mentioning a section feeds its full resolved contents to the assistant so it targets the exact block instead of guessing.
- Media gallery: tag your assets, search by tag, and edit tags inline from the gallery tiles.
- Media: when attaching an image to chat, a downscaled thumbnail is sent to the assistant for description while the full file still uploads.
- Chat polish: inline confirm before deleting a saved thread, a Retry affordance on failed turns, a single-line status footer with Debug moved to an overflow menu, and attached/referenced images shown inline as thumbnails in the message bubble.

## Fixes
- Chat: HTML comments and whitespace-only assistant parts no longer render as empty bubbles; long lines wrap in the code editor; tool-call pills are slimmer and wrap per row.
- Component develop view no longer causes a hydration mismatch (persisted view is read in an effect, not during render).
- Render: label expressions handle surrounding backticks automatically (legacy values heal on render, no migration needed).

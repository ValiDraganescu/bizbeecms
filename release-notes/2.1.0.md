# CMS v2.1.0

_2026-07-23 · changes since r-2.0.0_

## Features
- The AI assistant's model dropdown is now a real searchable picker: on sites with PM-curated models it lists exactly those aliases with their input/output prices (customer pricing, margin already applied); uncurated sites keep the full searchable catalog.
- Chat agents are now included in site export/import: an export carries every agent's full configuration (persona, model, welcome message, limits, and data-source/collection allowlists), and an import restores them. Conversation history and usage analytics are never exported and are never destroyed by an import. Older export files remain importable.

## Fixes
- The chat context meter now shows the correct context-window size on curated sites (it previously failed to resolve the model behind an alias).

## Other
- Project Manager: the AI-models settings page replaces the broken suggestion inputs with the same searchable picker — full OpenRouter catalog with search, capability filters, and per-model pricing; image-describe/image-generate purposes filter to capable models automatically.

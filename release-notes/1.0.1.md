# CMS v1.0.1

_2026-06-24 · changes since r-1.0.0_

## Fixes
- The AI assistant's debug tools that are restricted to BizBeeCMS SSO operators (export chat as JSON, the system-prompt versions editor) now appear correctly. Operators who signed in via "Sign in with BizbeeCMS" were being treated as non-SSO when their CMS account used their real email, so the controls stayed hidden.

## Notes
- Adds a `pm_user_id` column to the CMS `user` table (auto-applied on deploy). SSO operators are stamped with it on their next sign-in.

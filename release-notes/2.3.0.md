# CMS v2.3.0

_2026-07-24 · changes since r-2.2.1_

## Features
- **The AI assistant follows your selection in the Page Builder.** When you select a section — or a block inside one — the assistant now knows which one you're working on and sees its current contents. You can say "restyle this section" or "change this block" without @-mentioning it by name. Explicitly @-mentioning the same section still works and isn't duplicated.

# CMS v1.17.0

_2026-07-02 · changes since r-1.16.0_

## Features
- Icon sets: a per-site icon library with an editor in settings and an icon picker in the page builder, so pages can drop in named icons that render on the published site.
- Components: a "used-by" view — see which pages and components consume a given component before you edit it, so the blast radius of a change is visible up front.
- A built-in Language Switcher block you can place on a page; it renders the site's available content locales for visitors.
- Chat can produce transparent image cut-outs (background removed) alongside generated images, saved to your media library.

# CMS v1.0.0

_2026-06-24 · first `r-` release_

This is the first release on the new `r-<x.y.z>` tag scheme (`r` = release),
replacing the retired `cms-v*` series.

## Changes
- The admin sidebar now shows the running CMS release version (easier to confirm what's deployed).
- Removed the sidebar **AI Assistant** (the full-page `/admin/chat`). The floating AI assistant that opens on any admin page is unchanged.
- Removed the **Pages** admin section (the standalone page list/editor). Pages are still created and edited in the **Page Builder**; the public site, preview, and sitemap are unaffected.

## Notes
- Release tags are now `r-<x.y.z>`. The deployer and PM version picker read `r-*` (and still read legacy `cms-v*` so already-deployed sites resolve).

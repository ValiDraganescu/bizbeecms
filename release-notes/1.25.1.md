# CMS v1.25.1

_2026-07-07 · changes since r-1.25.0_

## Fixes
- Chat export (PM-SSO debug tool) now produces a complete, lossless dump of a conversation. Previously it saved only the reduced model-facing view — uploaded images, assistant reasoning, and full tool input/output were dropped. The download now also includes the raw transcript verbatim plus the actual request parameters and metadata (page context, prompt source, model context length).

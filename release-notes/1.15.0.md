# CMS v1.15.0

_2026-06-29 · changes since r-1.14.0_

## Features
- Page builder: per-field translation in the SEO/settings forms, so individual fields can be translated independently.
- Page builder: a gallery image picker for selecting media-library assets into fields.
- A configurable translation model setting (per-site translate-model), feeding the translate flow.

## Other
- Internal refactor: the page-builder shell and the render pipeline were split into smaller per-component / per-concern files (plan-tree, plan-section, plan-list, etc.), plus render hardening. No user-facing change from the refactor itself.

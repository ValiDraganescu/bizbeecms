# CMS v1.21.0

_2026-07-03 · changes since r-1.20.0_

## Features
- Page builder: every sizing control now has a rem/px unit picker, and every block's settings panel starts with standard spacing controls (per-side padding & margin). Existing pages keep their current look — legacy defaults are preserved.
- A more reliable number input across the page builder: fields can be fully cleared while typing (no more values snapping back mid-edit).
- Chat assistant context engineering: the assistant's system prompt now scopes to the tools available on the current page, and long lists (components, collections) are capped — cutting token cost substantially (e.g. data-sources admin ~17.7k → ~1.9k tokens) with no loss of capability. Old, cold chat threads are compacted on load so they no longer blow up context.
- Component authoring now lints generated HTML and scripts and reconciles CSS classes, catching malformed output before it ships.

## Fixes
- SSO: closed a `redirect_uri_mismatch` on login (the shared app-origin fix), verified live.
